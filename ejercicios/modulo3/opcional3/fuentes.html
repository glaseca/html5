Una de las tipografías más utilizadas de la API Google Fonts es "Open Sans", puedes visitar todas las fuentes que ofrecen desde el siguiente enlace: https://www.google.com/fonts.

Este texto aparece en su propia página:

To give developers a choice of high-quality fonts when using the Fonts API, we have created Google Fonts. There you can browse our catalog of available fonts, learn about the font designers who created them, and copy the code required to use them on your web page. The fonts in the directory are all released under open source licenses; you can use them on any non-commercial or commercial project.
